## 简介
这是一个使用 Rust 和 Piston 库实现的经典贪吃蛇游戏。玩家可以通过键盘控制蛇的移动，蛇吃到食物后会增长，目标是尽可能让蛇变长而不碰到自己的身体或窗口边界。

## 功能
### 已实现功能
- [x] 蛇的移动逻辑
- [x] 碰撞检测 (蛇头与食物)
- [x] 蛇的增长
- [x] 食物的随机生成

## 依赖
- Rust 编译器 (`rustc`)
- Cargo 包管理器
- Piston 库及其相关依赖

## 安装
```bash
git clone https://github.com/MJYEEE/Snake_rust.git
cd Snake_rust
cargo build
```

## 运行
```bash
cargo run
```
